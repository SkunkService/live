LIVE Stream - Documentation

Overview

This documentation provides details about the live streaming request page available at https://skunkservice.github.io/live/create. It allows users to submit a request to create a new streaming channel on the SkunkStream platform.

Getting Started

To start using the live streaming request page:

  1. Visit the request page at https://skunkservice.github.io/live/create.
  2. Fill in the required fields: Username Input and Streaming Partner Input.
  3. Click on the Create Request Button to submit your request.
  4. Wait for confirmation that your request has been received and is being processed.

Features

  • Username Input: Field for entering the Twitch or TikTok LIVE username.
  • Streaming Partner Input: Field for entering the streaming partner's username (e.g., tiktok@username, twitch/username).
  • Create Request Button: Submits the streaming request form.
  • Rules:
    • No NSFW content allowed.
    • No spamming requests.
    • No hacking on GitHub.
    • No bullying.
    • You must follow the rules.
    • You must have access to the TikTok LIVE or Twitch stream.

Request Process

Once a request is submitted:

  1. Your request will be reviewed by the SkunkStream team.
  2. You may receive a follow-up email or message if additional information is needed.
  3. After review, you will be notified of the status of your request and any next steps.
  4. If approved, your streaming channel will be created, and you will receive access details.

FAQs

What is SkunkStream?

SkunkStream is a platform for managing and creating streaming channels on Twitch and TikTok.

How do I submit a streaming request?

Fill in the username and streaming partner input fields on the request page and click the "Create Request" button.

What if I encounter issues?

If you experience problems with the request form or have other issues, please contact our support team at support@skunkservice.github.io.

Troubleshooting

If you face any issues with the live streaming request page, consider the following steps:

  • Check your internet connection: Ensure you have a stable connection to prevent submission issues.
  • Verify input fields: Make sure all required fields are filled in correctly.
  • Clear cache and cookies: Sometimes browser issues can be resolved by clearing cache and cookies.
  • Try a different browser: If problems persist, try accessing the page using a different browser.
